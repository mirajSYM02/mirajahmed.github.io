Project Title

"Education's impact on employability in Malaysia"


Following Data Analysis Steps & Project Progress


(1) Ask

⇒   How education impacted the employability of Malaysia from the year 2010 until 2019?
⇒   What level of study has higher rates of employability?
⇒   Male or female which gender leads the number of employment?
⇒   Which type of jobs are more popular among malaysians?
⇒   Which state has the higher employability rates?
⇒   Being a multi racial country, which race leads in terms of being employed?
⇒   Does uneducated people remains unemployed?
⇒   Which year has the highest employed and unemployed numbers?

(2) Prepare

⇒   A research was conducted to find solution. ⇒   Collection of several sets of data that related to employment resolved the stakeholder's queries.
⇒   As organization requires to use Microsoft excel and PowerBI as the main tools, so selected tools will be this.
⇒   Selected data was stored in Google Drive in case of any backup or sharing required.

(3) Process

⇒   The main data store that was being used is data.gov.my which is a reliable source, as it is government's data store.
⇒   To check the data quality, a check was being conducted.
⇒   Data inaccuracy was successfully handled through the quality check.


(4) Analyze

⇒   Combined the data sets into one big dataset to make a comprehensive story.
⇒   Added data into PowerBI to develop a dashboard to give it dynamic outlook.
⇒   The dashboard showcases the data in a dynamic order to resolve all the queries of the stakeholder.

(5) Share

⇒   Updated the dashboard user interface.
⇒   Added Map into the dashboard to make the states more visible and easy to understand.
⇒   Added small icons for better understanding.


(6) Act

⇒   Few more queries were being added to have a better outcome of the data at share phase.
⇒   Updated the dashboard as per the new queries.
⇒   Finalize the dashboard for publishing.
