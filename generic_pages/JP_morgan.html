Task 1: Creating Simple Data Analysis using Microsoft Excel on the Home Loan Dataset

Situation: The task involves creating a simple data analysis using Microsoft Excel on a home loan dataset. The dataset consists of approximately 500 data points that have been cleaned and processed to ensure suitability for analysis.

Task:

  • Utilize Microsoft Excel to perform data analysis on the home loan dataset.
  • Generate pivot charts to visualize the data and gain insights.

Actions:

  1. Clean and process the home loan dataset to prepare it for analysis.
  2. Utilize pivot charts in Microsoft Excel to create visual representations of the data.
  3. Analyze the generated charts to derive meaningful insights.


Results:

  • Identified a large number of borrowers with loan-to-value ratios within a specific range, suggesting a high demand for mortgages in that range. This presents a potential sales opportunity for lenders to offer competitive interest rates and terms, attracting these borrowers.
  • Discovered a significant number of borrowers falling within the low-income range. It is recommended that lenders consider offering more affordable loan products with lower interest rates and down payments to attract this customer segment.
  • Observed a higher number of second-time buyers compared to first-time buyers, indicating a potential trend in the market.
  • Determined that home values appear to be normally distributed, with an average value of approximately $430K.

Insights from the dashboard:

  • The most suitable age group to target for sales is 35-44.
  • The highest number of first-time buyers and second-time buyers also falls within the 35-44 age group.

  • Area code 27 has the lowest average loan-to-value ratio.
  • Area code 15 has the highest average appraised value of homes, which is $772,480.
